XML+XSLT->XHTML

西暦2000年前後、XMLはWebの技術として発展していた。特にXML+XSLTでXHTMLを生成するプロセスは未来技術として期待されていた。

しかしその後は期待されていた効果はなく、HTML5が浸透し始める。XSLTもXHTMLも主だって使われることなくXMLもJSONに置き換わろうとしている。

そんな中でXML隆盛から20年経った今、ついにXSLTのフレームワークが誕生する。Framexsだ。

FramexsはXHTMLをMVCの材料として合成を行うフレームワークである。
