Mozilla Developer Network JavaScript API

JavaScript

JavaScriptとはプログラミング言語の一種でウェブクライアントで実行されるほぼ唯一の言語として人気がある。

バージョンがありES3、ES5、ES6(ES2015)以降が大きな違いがある。

代表的なライブラリとしては

  • jQuery
  • Underscore
  • D3.js

がありCDNを通して利用することが多い。jsDeliver,cdnjsなどがある。

フレームワークがあり代表的なもとしては

  • Vue.js
  • React
  • Angular

がある。
