Framexsはクライアントサイド駆動のフレームワークであるため、その動作は静的サイトか動的サイトかを問わない。

例えばWordPressでFramexsの機能を使えるテーマを作っている。

Framexs for WordPress
https://github.com/inomoto-hironobu/framexs4wordpress

これをインストールするためにはGitを使う必要がある。

(wordpress)/wp-contents/theme/に移動して、下記のコマンドを実行する。

git clone https://github.com/inomoto-hironobu/framexs4wordpress.git

Framexsテーマを選ぶと管理画面でTheme Panelが選択できる。

このパネルではFramexsテーマのディレクトリを指定する。そうするとFramexsテーマが選べるようになる。
